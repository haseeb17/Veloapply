# Contributing

This repository holds public documentation and product information for VeloApply. The production application, backend services, and AI systems are maintained in private repositories and are not open for external contribution.

## How you can help

- Report unclear or incorrect documentation.
- Suggest improvements to public guides.
- File feature requests or bug reports through Issues.

For anything beyond a small text fix, please open an Issue first so we can confirm it fits the scope of this repository before you spend time on a pull request.
