# Security Policy

If you discover a security vulnerability in VeloApply, please report it privately rather than opening a public Issue.

## How to report

Email haseeb.kamran7@gmail.com with a description of the issue and the steps to reproduce it. We aim to acknowledge reports within a few business days.

## Scope

This applies to the VeloApply website, Chrome extension, and API. Please do not test against other users' accounts or data without permission.

Thank you for helping keep VeloApply and its users safe.
