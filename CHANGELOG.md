# Changelog

All notable changes to VeloApply are documented here.

## 2.9.1

- Chrome extension published on the Chrome Web Store.
- Autofill support across major ATS platforms, including Workday and Greenhouse.
- Queue based batch apply mode for reviewing multiple applications before submission.
- EEO field handling and custom dropdown support for common application forms.
